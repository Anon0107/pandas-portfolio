This repo has my pandas practice codes. Each file cleans data from csv or xlsx files in different ways.
data_cleaner1.py: Merging columns to remove duplicate values.
data_cleaner2.py: Add new columns by computing values from existing columns.
data_cleaner3.py: Extra basic data handling on top of data_cleaner3.py.
data_cleaner4.py: Cleans duplicates and fills missing values.
data_cleaner5.py: Computing summaries on top of data_cleaner4.py.
Titanic_analysis.ipynb: Notebook for Titanic data analysis using pandas and numpy.
